# Run Tests

Run the test suite for the project.

## Available Test Commands
- `npm run test:functions` - Run Firebase Cloud Functions tests
- `npm run test:e2e` - Run Playwright E2E tests
- `npx playwright test` - Run Playwright tests with visible browser

## Steps
1. Ask user which tests to run (functions, e2e, or all)
2. Run the selected tests
3. Report results summary
